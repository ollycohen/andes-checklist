# Andes Expedition Checklist

Interactive pre-departure checklist with priority matrix visualization.

- **List view**: Categories, checkboxes, 1–5 priority pips, date pickers, nested subtasks
- **Matrix view**: Drag-to-reorder Eisenhower-style scatter plot (Urgency × Importance)
- **Persistence**: localStorage — state survives page reloads
- **Zero dependencies**: Single HTML file, React via CDN

## Deploy

Hosted on GitHub Pages at [checklist.ollycohen.com](https://checklist.ollycohen.com).

## DNS Setup

Add a CNAME record for `checklist.ollycohen.com` pointing to `<your-github-username>.github.io`.
